Data Structures
=========================================

pyhydrophone implements one main structure, a :class:`Hydrophone` and its subclasses
* :class:`SoundTrap`
* :class:`Seiche`
* :class:`BruelKjaer`
* :class:`AMARG3`


Hydrophone
-------

The :class:`Hydrophone` represents an acoustic recorder and its properties and functions to read its output data
